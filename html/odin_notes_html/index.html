HTML Basics

Paragraphs, Bold, Italics, and Comments

  • This is a test to show that paragraphs be created using

    <p></p>

  • Now we will test the bold feature using the strong tag =

    <strong></strong>

  • We can also italicize words by using the 'em' tag. =

    <em></em>

  • Comments can be made by using the following:

    <!-- this is a comment -->

How to create lists

  • To create an unordered list, use:

    <ul></ul> and <li></li>
    • Item 1
    • Item 2

  • To create an ordered list, use:

    <ol></ol> and <li></li>
    1. Item 1
    2. Item 2

How to clone to GitHub

  1. create repo and copy SSH link
  2. open terminal and cd into folder that needs to be cloned
  3. git clone [SSH link]
  4. check status using: git remote show origin

How to sync with GitHub

  1. git status
  2. git add .
  3. git status
  4. git commit -m "[Comment describing change]"
  5. git push origin main
  6. git log: git log --oneline -5 --author -debugdeem
  7. Always branch off from main/master using git checkout -b feature/'nadeem'
